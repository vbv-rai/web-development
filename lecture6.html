Block vs Inline Elements

Block Elements

Block elements take up the full width available and start on a new line.

This is a div (block element). It starts on a new line and takes up the full width.

This is a paragraph (block element). It also starts on a new line and takes up the full width.

Inline Elements

Inline elements do not start on a new line. They only take up as much width as needed.
